# Zara-Fashion-Dashboard
🎯 Built my Excel Dashboard using real Zara Fashion data (20,252 rows)
📊 What I analyzed:

Sales Volume by Section (MAN vs WOMAN)
Season-wise performance
Promotion impact on sales
Product placement strategy
Top manufacturing countries

🔍 Key Insights:

WOMAN section drives 68% of total sales
Autumn is the peak season
Aisle placement generates highest sales volume

🛠 Tools used: Excel — Pivot Tables, Slicers, Charts, Descriptive Statistics
